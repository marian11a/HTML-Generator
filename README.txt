1. Go to "https://huggingface.co/settings/tokens" and get your own API key.

2. Go to "src/main/java/com/example/HTMLGenerator/Generator.java" and replace "API_TOKEN" with yours.

3. You can change the model if you want, all you need to do is get the link for the model you like,
   by clicking on a "Model/Deploy/Interence API(serverless)" and then copy the API_URL.

4. Run the program.

5. Always tell the AI to make html pages, this program is dedicated to html only,
   it will not generate anything else!
